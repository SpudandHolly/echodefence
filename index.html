EchoDepth as an AI Polygraph Alternative

EchoDepth is the leading UK-developed AI credibility assessment platform — built for defence, intelligence, and border security teams who need a defensible, evidence-based replacement for polygraph. Below are the questions we hear most from procurement, security leadership, and technical integration teams.

What is EchoDepth, and how does it fit within emotion recognition AI for defence?
EchoDepth is a camera-based emotion recognition AI platform developed in the UK by Cavefish AI, purpose-built for defence and security environments. As a defence AI system, it analyses facial muscle activations in real time to score the emotional and cognitive state of personnel — producing actionable output for insider threat, operator readiness, compliance, and deception detection use cases. EchoDepth is one of the only UK-developed emotion recognition AI systems designed specifically for classified and high-security environments, with full on-premise and air-gap deployment options as standard.
How does EchoDepth differ from standard facial expression analysis software?
Most facial expression analysis software maps images to a fixed set of basic emotions — happiness, sadness, anger — using broad classifier models trained on consumer datasets. EchoDepth operates at a more granular and scientifically grounded level: it extracts 43 individual facial Action Units (AUs) per frame using the Facial Action Coding System (FACS), then maps AU combinations to Valence-Arousal-Dominance (VAD) space. This means it detects nuanced states — suppressed stress, cognitive overload, micro-expressions preceding deception — that conventional facial expression analysis software cannot surface. The output is structured, timestamped numerical data suitable for security audit trails, not a sentiment emoji.
Can EchoDepth function as a stress detection AI for operator readiness monitoring?
Yes — stress detection and cognitive load monitoring are among EchoDepth's primary use cases in defence and security operations. For drone pilots, SOC analysts, control room operators, and intelligence reviewers, EchoDepth tracks facial Action Units associated with elevated arousal, cognitive overload, and fatigue onset in real time. This enables pre-mission go/no-go readiness scoring based on objective biometric data, live alerting when an operator's cognitive load exceeds safe thresholds, and post-incident timeline reconstruction mapping stress state against decision points. As a stress detection AI operating at approximately 700ms latency, EchoDepth is the only contact-free system capable of providing this level of cognitive load monitoring without wearables or disruption to workflow.
What makes EchoDepth a credible AI credibility assessment tool?
The case for EchoDepth as an AI polygraph alternative rests on two foundations: the scientific weakness of polygraph and the scientific strength of FACS. The US National Academy of Sciences found no scientific consensus supporting polygraph accuracy in 2003; polygraph false negative rates in controlled studies run as high as 47%. EchoDepth's AI credibility assessment approach instead uses peer-reviewed FACS methodology — 43 Action Units mapped to stress, suppression, and deception indicators — producing timestamped per-question output with confidence weightings. This gives investigators a documented, evidence-based record that can withstand legal and procurement scrutiny, which polygraph output cannot.
Is EchoDepth available as insider threat detection software in the UK?
EchoDepth is a UK-based insider threat detection software platform, developed and hosted by Cavefish AI in Cardiff, Wales, with UK data residency as the default for every deployment. For UK defence, critical national infrastructure, and financial security teams, EchoDepth provides continuous behavioural baseline profiling per individual, anomaly scoring when emotional patterns deviate significantly from an individual's established baseline, and direct SIEM integration for alerting and incident response workflows. The Ponemon Institute estimates the average cost of an insider incident at £3.2 million; EchoDepth is designed to surface the behavioural signals that precede incidents, not respond to them after the fact.
How does non-contact emotion detection work, and what hardware is required?
EchoDepth is a non-contact emotion detection system — the subject requires no physical interaction with any sensor, wearable, or equipment at any point. Camera-based physiological sensing uses a standard RGB camera feed to capture facial muscle movements, which are then processed locally by the EchoDepth engine. A minimum camera resolution of 720p is sufficient; existing CCTV infrastructure, interview room cameras, or laptop webcams all qualify. There is no requirement for infrared, thermal imaging, or specialist hardware. This non-contact approach is what makes EchoDepth operationally viable in interview settings, training environments, and control rooms where attaching physical sensors to personnel is impractical or inadmissible.
Is EchoDepth classified as a behavioural biometrics system?
EchoDepth operates within the behavioural biometrics category — specifically the sub-domain of facial behavioural biometrics. Where conventional biometrics (fingerprint, iris, face recognition) verify identity, behavioural biometrics systems like EchoDepth characterise state: the emotional, cognitive, and physiological condition of a known individual at a given moment. For procurement teams evaluating security technology, EchoDepth sits at the intersection of behavioural biometrics, emotion recognition AI, and physiological sensing — and is the only UK-developed platform in this space designed specifically for defence and classified environments rather than commercial HR or consumer applications.
How does EchoDepth support human reliability assessment in defence contexts?
Human reliability assessment (HRA) is a core methodology in NATO and MoD frameworks for evaluating whether personnel are fit to perform safety-critical or security-critical tasks. Traditional HRA is conducted periodically and relies on self-report, supervisor observation, and structured interviews. EchoDepth extends HRA into the real-time domain: it provides objective, continuous scoring of an individual's emotional readiness, cognitive load, and stress state — the same dimensions that HRA frameworks are designed to evaluate, but measured live and without reliance on self-disclosure. For defence teams operating under STANAG or JSP frameworks where human reliability is a documented safety requirement, EchoDepth provides the audit-ready, timestamped evidence layer that manual HRA processes cannot.
Can EchoDepth operate in a SCIF or air-gapped classified environment?
Yes. EchoDepth is fully deployable on-premise in Docker containers with zero external data transmission at any stage of the pipeline. It is engineered for SCIF-compatible classified environments — no cloud dependency, no telemetry, no outbound network calls. All inference runs locally on standard server hardware. UK data residency is the default for every deployment, and an air-gapped demo environment is available on request for vetted procurement and security teams. The system integrates with SIEM, LMS, and C2 platforms via REST API or WebSocket, with SDKs in Python and Node.js.
Is EchoDepth compliant with UK GDPR and defence data standards?
EchoDepth processes biometric data with pseudonymisation by default, role-based access controls, and full audit logging at every stage. All data is processed within UK borders as standard. The platform is aligned with NCSC principles, is on the ISO 27001 certification pathway, produces DSAT-compatible audit records for defence training compliance, and operates within the UK GDPR framework governing biometric data processing in security and law enforcement contexts.
